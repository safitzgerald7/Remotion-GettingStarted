# Design System for Tomato Lifecycle Video

## Colors
- Primary Green: #4CAF50
- Secondary Green: #66BB6A
- Dark Green: #388E3C
- Tomato Red: #FF6B35
- Tomato Orange: #FF5722
- Background: #FFFFFF
- Text Primary: #0D1117
- Text Secondary: #161B22
- Stake Brown: #8B5A2B
- Sun Yellow: #FFEB3B

## Typography
- Title Font: Playfair Display (Google Fonts)
- Title Size: 60px
- Title Weight: 700
- Title Letter Spacing: -1px
- Subtitle Font: Inter (Google Fonts)
- Subtitle Size: 26px
- Subtitle Weight: 400
- Subtitle Letter Spacing: 0.5px
- Subtitle Line Height: 1.4

## Spacing
- Base Unit: 4px
- Small: 8px
- Medium: 16px
- Large: 32px
- Extra Large: 60px

## Transitions
- Fade Duration: 15 frames (0.5s at 30fps)
- Easing: Easing.bezier(0.25, 0.46, 0.45, 0.94) for smooth organic motion
- Animation Duration: 1.5-2.5 seconds for scene elements
- Wind Sway: 2 seconds cycle

## Visual Elements
- Background: Solid white canvas with subtle texture or soft tonal overlays when needed, never transparent
- Plants: Gradient fills with drop shadows
- Tomatoes: Radial gradients with highlights
- Sun Rays: Radial gradient for lighting effect
- Layout: Horizontal and vertical centering for main scene composition
- Animations: Scale, translate, rotate with easing
