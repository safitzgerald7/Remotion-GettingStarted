# Design System

## Colors
- Primary: #FF6B35 (Tomato red)
- Secondary: #4ECDC4 (Teal)
- Background Gradient Start: #FFF8F0 (Warm white)
- Background Gradient End: #F5F0E8 (Light beige)
- Text: #2C3E50 (Dark blue-gray)

## Typography
- Font Family: 'Inter', sans-serif
- H1: 48px, 600 weight
- H2: 36px, 500 weight
- Body: 18px, 400 weight
- Caption: 14px, 400 weight

## Spacing
- Base: 4px
- Small: 8px
- Medium: 16px
- Large: 32px
- Extra Large: 64px

## Transitions
- Duration: 300ms (2.5 seconds at 30fps)
- Easing: ease-in-out (bezier 0.25, 0.46, 0.45, 0.94)
